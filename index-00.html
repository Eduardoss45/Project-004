Ecoturlabs é uma empresa privada com sede no Brasil focada no desenvolvimento da acessibilidade para pessoas com deficiencia auditiva e concientização da população. As empresas da Ecoturlabs incluem: Ecobra especializada em visitas a parques e zoologicos com serviços de tradução em libras no Brasil.

Propósito

Fazendo a diferença

Nossa equipe que prioriza trazer a possibilidade de pessoas com deficiência auditiva poder praticar atividades ecoturísticas com facilidade, segundo a Organização Mundial da Saúde (OMS), existem 500 milhões de surdos no mundo, pensando nisso mais afundo tivemos a ideia para melhorar a vida dessas pessoas, pois infelizmente e difícil ver algum serviço disponibilizar a possibilidade dessas pessoas poderem ler seus conteúdos através das libras. Hoje temos a possibilidade de poder mostrar histórias e fatos de vários locais através da tecnologia, pensando nisso criamos um site web que serve como guia de viagem ou visitas que tem como diferencial tradução para libras.

Mas o'que é libras?

É uma sigla de linguagem de sinais físicos que pessoas com deficiência auditiva ou de fala usam para se comunicar.

Mas o'que é ecoturismo?

É o segmento da atividade turística que utiliza, de forma sustentável, o patrimônio natural e cultural, incentiva sua conservação e busca a formação de uma consciência ambientalista por meio de interpretação do ambiente, promovendo bem-estar das populações.

Linguagem de Marcação de HiperTexto: HTML HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção básico da web. Define a estrutura do conteúdo da web.
Linguagem de Folhas de Estilos: CSS CSS é uma linguagem baseada em regras. Usada para especificar grupos de estilo que devem ser aplicados para elementos particulares ou grupos de elementos em sua página web.
Linguagem de Programação: JS JavaScript é uma linguagem de programação que permite implementar funções complexas em páginas ex: informação estática, conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráficos 2D/3D animado, etc. O JavaScript é a terceira camada do bolo das tecnologias padrões da web ja que as outras são o HTML e CSS.
